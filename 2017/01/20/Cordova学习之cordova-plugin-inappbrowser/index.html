Cordova学习之cordova-plugin-inappbrowser

发表于 2017-01-20 | 分类于 Cordova | 阅读次数:

以下所有步骤默认已安装node及平台SDK
官方文档

此插件提供了一个Web浏览器视图，当调用cordova.InAppBrowser.open()时显示。

var ref = cordova.InAppBrowser.open('http://apache.org', '_blank', 'location=yes');

cordova.InAppBrowser.open()函数被定义为window.open()函数的替代。现有的window.open()调用可以使用InAppBrowser窗口，通过替换window.open:

window.open = cordova.InAppBrowser.open;

InAppBrowser窗口的行为类似于标准Web浏览器，并且无法访问Cordova API。 因此，如果您需要加载第三方（不受信任的）内容，而不是将其加载到Cordova主视图中，则建议使用InAppBrowser。 InAppBrowser不受白名单的约束，也不在系统浏览器中打开链接。

InAppBrowser默认为用户提供自己的GUI控件（back，forward，done）。

deviceReady后可用。

document.addEventListener("deviceready", onDeviceReady, false);
function onDeviceReady() {
    console.log("window.open works well");
}

安装

cordova plugin add cordova-plugin-inappbrowser

如果你想让你的应用程序中的所有页面加载通过InAppBrowser，你可以hook在初始化期间window.open。 例如：

document.addEventListener("deviceready", onDeviceReady, false);
function onDeviceReady() {
    window.open = cordova.InAppBrowser.open;
}

cordova.InAppBrowser.open

在新的InAppBrowser实例，当前浏览器实例或系统浏览器中打开一个URL

var ref = cordova.InAppBrowser.open(url, target, options);

url

要加载的网址，如果URL包含Unicode字符，请调用encodeURI()。

target

加载网址的目标，默认为_self的可选参数。(String)

  • _self: 如果URL位于白名单中，则在Cordova WebView中打开，否则将在InAppBrowser中打开。
  • _blank: 在InAppBrowser中打开。
  • _system: 在系统的Web浏览器中打开。

options

InAppBrowser的选项。为可选值, 默认: location = yes.
选项字符串不能包含任何空格，每个键值对必须用逗号分隔。 功能名称不区分大小写。 所有平台支持以下值：

键名 类型 描述
location String 设置yes或者no打开或关闭InAppBrowser的地址栏

Android支持的值:

键名 类型 描述
hidden String 设置为yes以创建浏览器并加载页面，但不显示它。 加载完成时触发loadstop事件。 默认为no
clearcache String 设置为yes以在打开新窗口之前清除浏览器的cookie缓存
clearsessioncache String 设置为yes以在打开新窗口之前清除会话cookie缓存
zoom String 设置为yes以显示Android浏览器的缩放控件，设置为no以隐藏它们。 默认为yes
hardwareback String 设置为yes，以使用硬件后退按钮通过InAppBrowser的历史向后导航。 如果没有上一页，InAppBrowser将关闭。 默认值为yes，因此如果希望后退按钮只关闭InAppBrowser，则必须将其设置为no
mediaPlaybackRequiresUserAction String 设置为yes可防止HTML5音频或视频自动播放（默认为no）
shouldPauseOnSuspend String 设置为yes以使InAppBrowser WebView暂停/继续与应用程序停止背景音频（这可能需要避免像CB-11013中所述的Google Play问题）

iOS支持的值:

键名 类型 描述
closebuttoncaption String 设置为字符串以用作完成按钮的标题。 注意，你需要自己本地化这个值
disallowoverscroll String 设置为yes或no（默认值为no）。 打开/关闭UIWebViewBounce属性
hidden String 设置为yes以创建浏览器并加载页面，但不显示它。 加载完成时触发loadstop事件。 默认为no
clearcache String 设置为yes以在打开新窗口之前清除浏览器的cookie缓存
clearsessioncache String 设置为yes以在打开新窗口之前清除会话cookie缓存
toolbar String 设置为yes或no以为InAppBrowser打开或关闭工具栏（默认为yes）
enableViewportScale String 设置为yes或no，以防止通过元标记缩放视口（默认为no）
mediaPlaybackRequiresUserAction String 设置为yes可防止HTML5音频或视频自动播放（默认为no）
allowInlineMediaPlayback String 设置为yes或no，允许在线HTML5媒体播放，在浏览器窗口内显示，而不是设备特定的播放界面。 HTML的视频元素还必须包含webkit-playsinline属性（默认为no）
keyboardDisplayRequiresUserAction String 设置为yes或no以在表单元素通过JavaScript的focus()调用接收焦点时打开键盘（默认为yes）
suppressesIncrementalRendering String 设置为yes或no，直到在呈现之前接收到所有新的视图内容（默认为no）
presentationstyle String 设置弹出样式pagesheet，formsheet或fullscreen，默认fullscreen
transitionstyle String 设置过度样式fliphorizontal，crossdissolve或coververtical，默认coververtical
toolbarposition String 设置toolbar位置，bottom或top，默认为bottom

InAppBrowser

当target设置为’_blank’时，从对cordova.InAppBrowser.open的调用返回的对象。

方法：

  • addEventListener
  • removeEventListener
  • close
  • show
  • hide
  • executeScript
  • insertCSS

InAppBrowser.addEventListener

为InAppBrowser添加事件监听

ref.addEventListener(eventname, callback);
参数名 描述
eventname loadstart: 开始加载时触发.
loadstop: 停止加载时触发.
loaderror: 加载出错时触发.
exit: 当InAppBrowser的window关闭时触发.
callback 当事件触发时执行的函数。 函数传递一个InAppBrowserEvent对象作为参数 
var inAppBrowserRef;

function showHelp(url) {

    var target = "_blank";

    var options = "location=yes,hidden=yes";

    inAppBrowserRef = cordova.InAppBrowser.open(url, target, options);

    inAppBrowserRef.addEventListener('loadstart', loadStartCallBack);

    inAppBrowserRef.addEventListener('loadstop', loadStopCallBack);

    inAppBrowserRef.addEventListener('loaderror', loadErrorCallBack);

}

function loadStartCallBack() {

    $('#status-message').text("loading please wait ...");

}

function loadStopCallBack() {

    if (inAppBrowserRef != undefined) {

        inAppBrowserRef.insertCSS({ code: "body{font-size: 25px;" });

        $('#status-message').text("");

        inAppBrowserRef.show();
    }

}

function loadErrorCallBack(params) {

    $('#status-message').text("");

    var scriptErrorMesssage =
       "alert('Sorry we cannot open that page. Message from the server is : "
       + params.message + "');"

    inAppBrowserRef.executeScript({ code: scriptErrorMesssage }, executeScriptCallBack);

    inAppBrowserRef.close();

    inAppBrowserRef = undefined;

}

function executeScriptCallBack(params) {

    if (params[0] == null) {

        $('#status-message').text(
           "Sorry we couldn't open that page. Message from the server is : '"
           + params.message + "'");
    }

}

InAppBrowserEvent Properties

loadstart 和 loaderror 事件在浏览器中不会触发。

属性 类型 描述
type String 事件名称
url String 加载的URL
code Number 错误代码，只在loaderror中
message String 错误信息，只在loaderror中 
var ref = cordova.InAppBrowser.open('http://apache.org', '_blank', 'location=yes');
ref.addEventListener('loadstart', function(event) { alert(event.url); });

InAppBrowser.removeEventListener

移除事件监听。

ref.removeEventListener(eventname, callback);

var ref = cordova.InAppBrowser.open('http://apache.org', '_blank', 'location=yes');
var myCallback = function(event) { alert(event.url); }
ref.addEventListener('loadstart', myCallback);
ref.removeEventListener('loadstart', myCallback);

InAppBrowser.close

关闭InAppBrowser window。

ref.close();

InAppBrowser.show

将已打开并且是隐藏状态的InAppBrowser窗口显示出来。 如果InAppBrowser已可见，则调用此方法无效。

var ref = cordova.InAppBrowser.open('http://apache.org', '_blank', 'hidden=yes');
// some time later...
ref.show();

InAppBrowser.hide

隐藏InAppBrowser窗口。 如果InAppBrowser已隐藏，则调用此方法无效。

var ref = cordova.InAppBrowser.open('http://apache.org', '_blank');
// some time later...
ref.hide();

InAppBrowser.executeScript

将JavaScript代码注入InAppBrowser窗口。

ref.executeScript(details, callback);
属性 类型 描述
injectDetails Object 要执行的代码，指定文件或代码
file String 要注入的脚本的URL
code String 要注入的脚本的文本
callback function 在JavaScript代码注入后执行的回调函数。如果注入的脚本是类型代码，则回调将使用单个参数执行，该参数是包含在Array中的脚本的返回值。 对于多行脚本，这是最后一个语句的返回值，或最后一个计算的表达式 
var ref = cordova.InAppBrowser.open('http://apache.org', '_blank', 'location=yes');
ref.addEventListener('loadstop', function() {
    ref.executeScript({file: "myscript.js"});
});

浏览器中仅支持代码不支持文件。

InAppBrowser.insertCSS

在InAppBrowser窗口中注入CSS。

var ref = cordova.InAppBrowser.open('http://apache.org', '_blank', 'location=yes');
ref.addEventListener('loadstop', function() {
    ref.insertCSS({file: "mystyles.css"});
});
属性 类型 描述
injectDetails Object 要执行的代码，指定文件或代码
file String 要注入的stylesheet的URL
code String 要注入的stylesheet的文本
callback function 在CSS代码注入后执行的回调函数。
